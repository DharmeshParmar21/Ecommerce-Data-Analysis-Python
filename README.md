## Project Overview
This Ecommerce Data Analysis project focuses on understanding sales performance, customer behavior, and profit trends using Python-based Exploratory Data Analysis (EDA).

The analysis includes:
- Data cleaning  
- Data preprocessing  
- Sales & profit trend analysis  
- Category & sub-category performance  
- Customer segment insights  
- Visualizations for decision-making  

All insights are derived using Python, as documented in the Jupyter Notebook and PDF report.

## Key Insights
- **Technology** category generates the highest sales  
- **Furniture and Office Supplies** follow as strong contributors  
- **Phones, Chairs, and Machines** are top-performing sub-categories  
- **Fasteners and Labels** show the lowest profitability  
- **Profit increases significantly** toward year-end (Novâ€“Dec)  
- **Consumer segment** provides the highest sales & profit  
- **Certain states** outperform others in revenue and order volume
